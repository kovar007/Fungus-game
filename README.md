# Point-and-Click Game Demo

## About the project

This is a 2D point and click adventure game developed in Java (NetBeans) using Swing.  
The player can move through scenes, switch between them, collect and store items, and interact with the environment.



## Features

- Multiple playable scenes
- Main menu, pause menu, and context menus
- Inventory system
- Dialogue system
- Character animations (idle, walking, triggered animations)
- Walkable area system using masks



## Controls

- **Mouse**
  - Moving the character by clicking on walkable areas
  - Interacting with objects through a context menu
  - Opening the inventory using the bag icon
  - Navigating through menus and scene transitions

- **Space**
  - Advance dialogue text

- **ESC**
  - Open pause menu

## Limitations

- No save/load system
- Fixed window size
- No audio (music or sound effects not implemented yet)
- Object hitboxes are rectangular and not pixel-perfect
